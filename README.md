## Description
Web Service para receber e enviar requisições de um app do slack que irá cuidar de registrar as reuniões diárias das equipes scrum.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
